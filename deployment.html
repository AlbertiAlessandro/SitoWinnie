How to Deploy a Project from PHPStorm

  1. Go to the Tools menu --> Deployment --> Configuration
  2. Click the button with the + symbol to add a new deployment configuration
  3. Give a meaningful name to the created configuration
  4. Select Type as SFTP
  5. If it's not already present, add an SSH configuration by clicking the three dots...
  6. In the Host field, enter Winnie's address: 10.4.55.123
  7. In the Username field, enter the username provided at the beginning of the year
  8. Leave Authentication Type as Password
  9. Test the connection with Test connection, if everything is fine, or make necessary modifications if needed
  10. Optionally, save the password
  11. Click OK and return to the initial screen
  12. Set the Root Path of the project by clicking the folder icon and navigating to your folder located at /home/username/public_html/subfolder
  13. In WebServer URL, enter the URL of your project on Winnie, for example, ~albe
  14. Switch to the Mappings tab
  15. Leave LocalPath as the default
  16. In the other two fields, unless there are specific requirements, enter / and /
  17. Finally, click OK, and you will have a deployment configuration
  18. For convenience, in the Tools menu --> Deployment, you can check Automatic upload
