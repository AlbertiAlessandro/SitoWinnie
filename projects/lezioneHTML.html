Introduction to HTML

The h Tag

The h tags stand for headings and are numbered from 1 to 6 in order of importance.


Paragraphs

You can use the p tag or the div tag. The result and usage of these tags are slightly different, but they are both block-level tags.

This is a block of code enclosed by a div.

Hyperlinks

Hyperlinks allow you to navigate when clicked with the mouse. The tag used in this case is a, and it's an inline tag. It can link to any external or internal page on the website, or even to other parts of the document.

For example, this is a link to our institute.

Tags for Inserting Images

To insert images, you use the img tag, which is always inline like the tag for creating links. The functionality is very similar to that of links, in the sense that you need to specify the resource, which is the image in this case.

This is an example where I include an image from the internet.

Bulleted and Numbered Lists

Lists are created using the ul tag (unordered, bulleted list) or the ol tag (ordered, numbered list). Within these tags, list items are added using the li tag.
